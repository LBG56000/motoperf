# Lancement du projet

## Init

```bash
cd backend
npm install
```

```bash
cd frontend
npm install
```

## Lancer l'appli

Se placer à la racine du projet

```bash
make start
```

## Supprimer les volumes

Redémmarrer l'appli

```bash
make clear
```

## Supprimer les volumes

Se placer à la racine du projet

```bash
make clear-all
```
